# System Architecture Overview

## 1. Core Components

### Frontend Layer
- Framework: Next.js 14 (App Router)
- UI: ShadCN/UI components
- State: Zustand
- File Handling: AWS S3 integration
- Real-time: Server-sent events

### Backend Layer
- API: FastAPI
- Database: 
  - PostgreSQL (structured data)
  - Pinecone (vector storage)
  - Redis (caching/queues)
- File Storage: AWS S3

### AI/ML Layer
A. Multi-Agent System
- Brief Analysis Agent (GPT-4V + CLIP)
- Style Intelligence Agent (Fashion-CLIP)
- Creative Exploration Agent (SDXL Turbo + ControlNet)
- Quality Assurance Agent (Custom ResNet)

B. Memory Systems
- Working Memory: Redis
- Episodic Memory: Neo4j
- Semantic Memory: Milvus
- Procedural Memory: PostgreSQL
